# Changelog

## [0.1.0] - 2025-05-14

### Added

- Initial release of **BaseBuilder** — AI Agent Toolkit on Base
- `ChatPanel`: Conversational UI with LangChain ReAct agent + OpenAI GPT-4o-mini
- `WalletPanel`: CDP wallet provider guide & network selector
- `ActionsPanel`: Searchable explorer for 15+ AgentKit action providers (40+ actions)
- `Header`: App header with live network status indicator
- `Sidebar`: Navigation between Chat, Wallet, and Actions panels
- `/api/agent`: REST endpoint powering the AI agent (AgentKit + LangGraph backend)
- `CdpEvmWalletProvider` integration for Base Sepolia and Base Mainnet
- Action providers: Wallet, WETH, ERC-20, ERC-721, Basename, Pyth, CDP API
- `.env.example` for quick environment setup
- Dark UI theme with Base blue (`#0052ff`) accent
